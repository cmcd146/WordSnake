Directions

  • The goal is to gain points by entering valid english words.
  • Besides the first word, the next word starts with the last letter from the previous word.
  • Game will end if player gets three strikes or gets trapped.
  • Just type normally to add and remove characters.
  • Press enter to submit word.
  • You can change direction using arrow keys. You can only move perpendicular to previous valid word.
